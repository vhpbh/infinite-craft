⚙️ הגדרות מערכת — יצירה אינסופית

דף זה מיועד רק למנהל. אין להעלות אותו לאינטרנט יחד עם index.html — הוא שומר את הגדרות החיבור בדפדפן שלכם בלבד (localStorage), הן לא נשלחות לשום מקום אחר.

🔌 חיבור ל-Supabase

ה-URL וה-anon key נמצאים ב-Supabase תחת Project Settings → API. מפתח ה-anon בטוח לחשיפה בצד לקוח (יש לו הרשאות קריאה בלבד בזכות ה-RLS שהגדרנו).
מפתח ה-service_role נדרש כאן כדי שכפתורי המחיקה למטה יעבדו (המפתח הרגיל מוגבל לקריאה בלבד). זהו מפתח סודי מלא הרשאות — אל תשתפו את דף הניהול הזה עם אף אחד, ואל תעלו אותו לאינטרנט.

🔑 מפתחות שרת (Edge Function Secrets)

מפתח ה-Gemini ומפתח ה-service_role לא מוזנים כאן — הם רגישים מדי לשבת בדפדפן. יש להגדיר אותם כ-secrets בפרויקט ה-Supabase שלכם דרך שורת הפקודה, כדי שיישארו מוסתרים לגמרי בצד השרת:
supabase secrets set GEMINI_API_KEY=המפתח_שלך_מ-Google_AI_Studio supabase secrets set SUPABASE_SERVICE_ROLE_KEY=המפתח_מ-Project_Settings_API supabase functions deploy combine
את מפתח ה-Gemini מקבלים חינם ב-aistudio.google.com/apikey. את ה-service_role key מוצאים באותו עמוד כמו ה-anon key, תחת Project Settings → API (שדה "service_role").

🗄️ הקמת בסיס הנתונים

הריצו את הקובץ supabase/schema.sql שסופק לכם פעם אחת, דרך Supabase → SQL Editor → New query.
